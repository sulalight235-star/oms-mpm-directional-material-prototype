# OMS-MPM Directional Material Prototype

OMS-MPM 定向物料生成原型项目。

## 项目说明

本仓库包含 OMS-MPM 定向物料相关的原型页面与需求文档，主要用于本地预览、评审和交互确认。

## 文件结构

- `OMS-MPM 定向物料生成.md`：需求文档
- `oms-mpm-物料工作台原型.html`：物料工作台原型
- `待我审核.html`：待我审核页面原型
- `审核结果.html`：审核结果页面原型
- `物料审核详情.html`：物料审核详情页面原型

## 本地运行

本项目为纯静态 HTML 原型，无需安装依赖。

### 方式一：直接打开

直接用浏览器打开以下任意文件即可：

- `oms-mpm-物料工作台原型.html`
- `待我审核.html`
- `审核结果.html`
- `物料审核详情.html`

### 方式二：使用本地静态服务

如果你希望通过本地服务预览，可在仓库目录执行任一方式：

```bash
python -m http.server 8000
```

然后在浏览器访问：

```text
http://localhost:8000/
```
